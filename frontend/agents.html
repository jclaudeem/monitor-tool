Settings
An agent is a small Node.js process you install on any machine inside a private network. It pings devices locally and pushes results to this server. The server must be reachable from the agent's network (via internet, VPN, or port forwarding).

Setup instructions

1 Copy the agent/ folder to a machine inside the target network that has internet access back to this server.
2 Install Node.js on that machine, then run npm install inside the agent/ folder.
3 Copy config.example.json → config.json and fill in your server URL and the API key above.
4 Run node agent.js (or install as a service — see README). The agent will check in immediately.
5 Go to Devices and assign devices to this agent. The agent will start monitoring them on the next poll cycle.
